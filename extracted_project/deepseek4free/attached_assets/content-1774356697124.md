# DeepSeek credentials [\#](https://docs.n8n.io/integrations/builtin/credentials/deepseek/\#deepseek-credentials "Permanent link")

You can use these credentials to authenticate the following nodes:

- [Chat DeepSeek](https://docs.n8n.io/integrations/builtin/cluster-nodes/sub-nodes/n8n-nodes-langchain.lmchatdeepseek/)

## Prerequisites [\#](https://docs.n8n.io/integrations/builtin/credentials/deepseek/\#prerequisites "Permanent link")

Create a [DeepSeek](https://platform.deepseek.com/sign_up) account.

## Supported authentication methods [\#](https://docs.n8n.io/integrations/builtin/credentials/deepseek/\#supported-authentication-methods "Permanent link")

- API key

## Related resources [\#](https://docs.n8n.io/integrations/builtin/credentials/deepseek/\#related-resources "Permanent link")

Refer to [DeepSeek's API documentation](https://api-docs.deepseek.com/api/deepseek-api) for more information about the service.

## Using API key [\#](https://docs.n8n.io/integrations/builtin/credentials/deepseek/\#using-api-key "Permanent link")

To configure this credential, you'll need:

- An **API Key**

To generate your API Key:

1. Login to your DeepSeek account or [create](https://platform.deepseek.com/sign_up) an account.
2. Open your [API keys](https://platform.deepseek.com/api_keys) page.
3. Select **Create new secret key** to create an API key, optionally naming the key.
4. Copy your key and add it as the **API Key** in n8n.

Refer to the [Your First API Call](https://api-docs.deepseek.com/) page for more information.
